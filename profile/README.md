# AutoPhD-org
## Overview
`AutoPhd-org` is an organization that aims to develop LLM-driven tools to assist various aspects of the PhD research process.

## Projects
- [ppagent](https://github.com/AutoPhd-org/your-paper-reading-agent): A highly configurable tool that helps researchers read and summarize academic papers based on personal preferences.
- [schedule-everything](https://github.com/AutoPhd-org/schedule-everything): A cli tool that helps researchers\AI agents manage their schedules and deadlines.
- [cv-generator](https://github.com/AutoPhd-org/your-cv-agent): Generates a high-quality CV based on the user's academic and professional background.
- More projects coming soon...
- [auto-phd.sh](https://github.com/AutoPhd-org/auto-phd.sh): A shell script that installs and configures all the necessary tools from the organization in one go.
